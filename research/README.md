## 目录说明

- **调研报告**：存放技术调研、工具对比、方案选型等文档
- **研究笔记**：存放实验记录、问题分析、优化思路等文档
- **历史文档**：存放项目发展过程中的重要历史记录

## 与其他目录的区别

| 目录 | 用途 | 内容类型 |
|------|------|---------|
| `docs/` | 技术文档 | 项目架构、API 文档、使用指南、数据格式规范 |
| `research/` | 调研报告 | 技术调研、工具对比、方案选型、研究笔记 |
| `configs/` | 配置文件 | YAML 配置文件 |
| `scripts/` | 运行脚本 | 数据生成、测试、检查等可执行脚本 |
