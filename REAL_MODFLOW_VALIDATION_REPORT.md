## 合成数据 vs 真实 MODFLOW

### 为什么合成数据 R² 高但不可信？

#### 合成数据生成函数

```python
# 这是简化的线性关系！
base_level = strt_norm * 5 + hk_norm * 2
trend = (pumping_norm * 0.01 + rch_norm * 0.005) * n_timesteps
seasonal = sy_norm * 2 * np.sin(2 * np.pi * t / 365)
noise = np.random.normal(0, 0.3)

timeseries = base_level + trend + seasonal + noise
```

**问题**:
- 这是**线性组合** + 噪声
- 模型只需学习几个权重
- 不代表真实 MODFLOW 的复杂非线性 PDE

---

#### 真实 MODFLOW

```
∂h/∂t = ∇·(K∇h) + W  （达西定律 + 连续性方程）
```

**特点**:
- 复杂的非线性偏微分方程
- 空间-时间耦合
- 边界条件影响
- 参数之间的非线性交互

---

### 参数-时序相关性对比

| 参数 | 合成数据相关性 | 真实 MODFLOW 相关性 | 差异 |
|------|---------------|-------------------|------|
| hk (渗透系数) | 0.3430 | 0.0858 | 4× 更强 |
| pumping (抽水) | 0.3179 | -0.0159 | 符号都反了！ |
| strt (初始水位) | 0.8578 | 0.9154 | 相似 |
| rch (补给) | 0.1229 | 0.0258 | 5× 更强 |

**结论**:
- 合成数据：多个参数都有中等相关性（容易学习）
- 真实 MODFLOW：只有 strt 强相关，其他参数复杂耦合
